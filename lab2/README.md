# Исследовательский анализ данных

## Описание проекта

Данный проект представляет собой исследовательский анализ двух датасетов:
1. **Общая часть**: Анализ датасета `mpg` (автомобильная статистика) из библиотеки Seaborn
2. **Самостоятельная часть**: Анализ футбольных данных UEFA Champions League 2016-2022

В отличие от первой лабораторной работы, здесь **не требуется** создание базы данных или графического интерфейса. Вся работа выполняется в Jupyter Notebook с использованием библиотек Python для анализа данных.

## Структура главных файлов проекта

- general_tasks.ipynb:          Анализ датасета mpg
- solo_tasks.ipynb:             Анализ футбольных данных
- unifyed_data_provider.py:     Скрипт для обработки футбольных данных

## Требования

- Python 3.10+ (работа проверена на версии 3.10.5)
- Библиотеки Python: pandas, numpy, matplotlib, seaborn, scipy
- Для анализа футбольных данных требуется файл Excel с исходными данными

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install pandas numpy matplotlib seaborn scipy
```

### 2. Подготовка данных

#### Для общей части (mpg датасет):
Датасет автоматически загружается через библиотеку Seaborn. Дополнительных файлов не требуется.

#### Для самостоятельной части (футбольные данные):
1. Скачайте файл `UEFA Champions League 2016-2022 Data 3.xlsx` [отсюда](https://www.kaggle.com/datasets/cbxkgl/uefa-champions-league-2016-2022-data)
2. Поместите файл в корневую папку проекта

### 3. Запуск анализа

#### Общая часть (mpg):
Откройте файл `general_tasks.ipynb` в Jupyter Notebook и выполните ячейки.

#### Самостоятельная часть (футбольные данные):
1. Сначала запустите скрипт `unifyed_data_provider.py` для обработки Excel-файла.
2. Выполните ячейки в `solo_tasks.ipynb`.

## Описание компонентов

### 1. Общая часть (general_tasks.ipynb)

Анализ датасета `mpg`, содержащего информацию о характеристиках автомобилей:

**Выполненные задачи:**
- Разведочный анализ данных (EDA)
- Статистика для числовых и категориальных переменных
- Проверка двух статистических гипотез:
  1. Зависимость ускорения от веса автомобиля
  2. Сравнение мощности автомобилей из США и других стран
- Кодирование категориальных переменных
- Построение корреляционной матрицы
- Реализация градиентного спуска (обычного и стохастического)

### 2. Скрипт unifyed_data_provider.py

Вспомогательный скрипт для обработки футбольных данных:

**Функциональность:**
- Загрузка и объединение таблиц из Excel-файла (goals, matches, players)
- Обработка и нормализация дат
- Создание новых признаков:
  - `IS_PENALTY`: является ли гол пенальти
  - `FIRST_HALF_GOAL`: забит ли гол в первом тайме
  - `PLAYER_AGE`: возраст игрока на момент матча
  - `TOTAL_MATCH_GOALS`: общее количество голов в матче
  - `HOME_GOAL`: забит ли гол домашней командой

### 3. Самостоятельная часть (solo_tasks.ipynb)

Анализ обработанных футбольных данных:

**Выполненные задачи:**
- Преобразование категориальных переменных в числовые (One-Hot Encoding)
- Разведочный анализ полученного датасета
- Проверка двух статистических гипотез:
  1. Сравнение количества голов в первом и втором таймах
  2. Влияние возраста игрока на вероятность забить пенальти
- Корреляционный анализ между признаками и целевой переменной

## Примечания

- Для работы с футбольными данными необходимо наличие исходного Excel-файла
- Все графики строятся автоматически при выполнении ячеек ноутбука
- Статистические тесты используют уровень значимости alpha = 0.05
- Код содержит комментарии для понимания логики анализа
